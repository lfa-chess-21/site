Welcome to the LFA Chess Club API Documentation

The goal of this page is to explain each route one by one

Route /api/routes

Method : GET

Gives a List of the API routes with the documentation of each route

Route /api/login

Method : POST

  • ARGUMENT : METHOD=POST NAME='username'
  • ARGUMENT : METHOD=POST NAME='password'
  • ARGUMENT : METHOD=GET NAME='next' OPTIONNAL : redirects to a certain url otherwise gives a JSON value

Request to login to an account

Route /api/logout

Method : POST or GET

  • ARGUMENT : METHOD=GET NAME='next' OPTIONNAL : redirects to a certain url otherwise gives a JSON value

Request to logout from an account
